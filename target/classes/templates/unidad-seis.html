Etiquetas y comparaciones

En el lenguaje ensamblador, las etiquetas son utilizadas para identificar ubicaciones específicas en el código. Las comparaciones permiten realizar evaluaciones de igualdad, diferencia, mayor que, menor que, etc.

Etiqueta Descripción Ejemplo
MOV Transfiere el contenido de un origen a un destino MOV AX, BX
ADD Suma dos valores y almacena el resultado en un destino ADD CX, DX
SUB Resta un valor de otro y almacena el resultado en un destino SUB BX, AX
CMP Compara dos valores y actualiza los flags de estado CMP AX, 0

Condicionales e incondicionales

Las instrucciones condicionales permiten ejecutar un bloque de código basado en una condición. Las instrucciones incondicionales se ejecutan sin ninguna condición adicional.

Condición Descripción Ejemplo
JMP Salto incondicional a una ubicación específica en el programa JMP L1
JZ Salto si el resultado de la comparación es igual a cero JZ L2
JNZ Salto si el resultado de la comparación no es igual a cero JNZ L3

Procedimientos y operaciones básicas

Los procedimientos son bloques de código reutilizables que pueden ser invocados desde otras partes del programa. Las operaciones básicas incluyen suma, resta, multiplicación y división.

Procedimiento Descripción Ejemplo
CALL Llama a un procedimiento o subrutina CALL Subrutina1
RET Retorna de una subrutina RET
INC Incrementa el valor de un registro o memoria en 1 INC AX

Simulación de while, do-while y for

Las instrucciones de repetición como while, do-while y for permiten ejecutar un bloque de código múltiples veces hasta que se cumpla una condición específica.

Instrucción Descripción Ejemplo
WHILE Ejecuta un bloque de código mientras se cumple una condición WHILE CX > 0
DO-WHILE Ejecuta un bloque de código al menos una vez y luego comprueba una condición DO
FOR Ejecuta un bloque de código un número específico de veces FOR CX, 1, 10

Instrucciones repetitivas

Las instrucciones repetitivas permiten repetir un bloque de código un número determinado de veces. Esto es útil para realizar tareas iterativas como recorrer arreglos o realizar cálculos repetidos.

Instrucción Descripción Ejemplo
LOOP Repite un bloque de código un número específico de veces LOOP L1
REPEAT Repite un bloque de código hasta que se cumpla una condición REPEAT
FOR EACH Itera sobre una colección de elementos y ejecuta un bloque de código para cada elemento FOR EACH elemento IN colección
